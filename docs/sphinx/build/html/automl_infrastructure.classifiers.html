class automl_infrastructure.classifiers.ensemble_classifier.EnsembleClassifier(name, input_models, ensemble_model, ensemble_extra_features=None)¶

Bases: automl_infrastructure.classifiers.base.Classifier

The EnsembleClassifier class represents a complex classifier that consists of one or more sub-classifiers. The way the complex classifier works is:

  • applying every sub-classifier on the dataset

  • gather from every sub-classifiers the classes probabilities prediction, and apply ensemble model on them with extra features (optional)

Parameters

  • name (str) – the name of the classifier.

  • input_models (list of automl_infrastructure.classifiers.base.Classifier) – list of sub-classifiers to aggregate their predictions.

  • ensemble_model (automl_infrastructure.classifiers.base.Classifier) – the aggregator classifier on top the sub-classifiers and possibly extra features.

  • ensemble_extra_features (list of str ,optional) – list of extra features that the ensemble model should work on.
