class automl_infrastructure.classifiers.adapters.keras_classifier.KerasClassifierAdapter(name, keras_classifier, features_cols=None)¶

Bases: automl_infrastructure.classifiers.base.BasicClassifier

The KerasClassifierAdapter implements an adapter (wrapper) for keras models, to match the automl_infrastructure.classifiers.base.Classifier contract.

Parameters

  • name (str) – name of the classifier (e.g. ‘lr1).

  • keras_classifier – keras classifier to wrap.

  • features_cols (list of str ,optional) – features list that the classifier should work on, while ignoring other features.
