automl_infrastructure.experiment.metrics.base module¶

class automl_infrastructure.experiment.metrics.base.Metric(is_grouped=False, weighted=True)¶

Bases: abc.ABC

The class Metric defines an interface for metrics. Formally speaking, metric may: - aggregate classes scores to a single score (e.g. average precision of all classes). - if aggregation was required, weather to weight classes according to their

size (e.g. weighted average precision of all classes).

Parameters

  • is_grouped (bool ,optional) – weather to aggregate classes score to single value.

  • weighted (bool ,optional) – weather to use weights (according to classes size) during aggregation

abstract measure(y_true, classifier_prediction)¶

The method receives true labels and prediction (class prediction along all classes probabilities) and returns score for each class if aggregation wasn’t asked by the user, and one score otherwise.

Parameters

  • y_true (pandas.Series or list) – true labels

  • classifier_prediction (list of automl_infrastructure.classifiers.base.ClassifierPrediction) – class prediction with classes probabilities list

Returns

if aggregation wasn’t asked by the user, list of classes scores ordered by classes nature sorting (e.g. alphabetically for classes of type string) and one score otherwise (weighted if was asked by the user).
